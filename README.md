# Farm

2D top-down life RPG prototype.

## Concept

Start from a small home, gather materials from the farm and forest, enter dungeons for monster and crystal materials, then craft useful items at the guild hall or workshop.

## First Goal

Build a small playable MVP:

- Move a player character in a top-down map
- Interact with animals, trees, rocks, and crystals
- Fight simple monsters with a sword
- Collect item drops
- Open inventory and crafting UI
- Craft a dungeon key
- Defeat a final orc encounter

## Asset Direction

Main visual style uses top-down pixel art assets from CraftPix and audio from Kenney CC0 sound packs.
